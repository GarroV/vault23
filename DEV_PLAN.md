# План разработки — Task Assistant Bot

> Стек: Supabase (Postgres + pgvector + Edge Functions + Storage) · TypeScript/Deno · Telegram Bot API · OpenAI
> Модель работы: Claude — архитектура, адаптация, проверка. Gemini (корпоративный, без лимитов) — генерация объёмных/шаблонных блоков по готовым спецификациям.

---

## Как читать пометки

- 🟢 **GEMINI** — объёмный/шаблонный блок с чёткими границами. Отдаём Gemini по меткому промту, Claude проверяет и стыкует.
- 🟡 **СОВМЕСТНО** — Gemini может сгенерить черновик, но Claude существенно дорабатывает (логика важнее объёма).
- 🔴 **CLAUDE** — нервная система / безопасность. Пишем и проверяем вручную, Gemini НЕ отдаём.

Принцип отдачи Gemini: **объём и шаблонность — да, хитрая логика и безопасность — нет.** На шаблонных блоках экономия реальна; на сложной логике проверка чужого кода стоит столько же, сколько написать самому.

---

## Предусловие (делаем до любого кода)

Это «пакет контекста» — единый источник правды, который вкладывается в каждый промт для Gemini, чтобы блоки стыковались.

| Артефакт | Кто | Зачем |
|---|---|---|
| `DATA_MODEL.md` — точные таблицы, поля, связи, дефолты | 🔴 CLAUDE | Без него Gemini выдумает свои структуры — куски не сойдутся |
| `MODULE_CONTRACT.md` — как модуль регистрируется и общается с ядром | 🔴 CLAUDE | Единый интерфейс для всех модулей |
| `CONVENTIONS.md` — именование, формат ошибок, стиль | 🔴 CLAUDE | Чтобы сгенерированный код выглядел единообразно |

## Сводка: что реально уходит Gemini

**Хороших кандидатов на Gemini (🟢) больше всего в:** CRUD-операциях, справочниках, миграциях SQL, обёртках над API (Whisper, embeddings), форматировании сообщений, агрегирующих запросах. Это объёмная рутина — там экономия максимальна.

**Остаётся на Claude (🔴):** всё про безопасность (RLS, лимиты, права), нервную систему (роутер, контракт, сессии), ключевые принципы (review-gate), и тонкую логику (конфликты календаря, NLU).

**Конвейер на каждый 🟢/🟡 блок:**
1. Claude пишет меткий промт с вложенным пакетом контекста (`DATA_MODEL` + `MODULE_CONTRACT` + `CONVENTIONS`).
2. Gemini генерит блок.
3. Claude адаптирует, проверяет стыковку и безопасность, интегрирует.
